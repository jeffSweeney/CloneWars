# CalculatorClone

Clone of the iPhone iOS 17 Calculator app, with my [CloneWarsIntro package](https://github.com/jeffSweeney/SweeneyCloneWarsIntro) injected in.

## Tech Stack

Built for iOS 18+, EntertainMe leverages this core tech stack:
- Swift & SwiftUI
- MVVM Architecture
- SPM
